简介：
冒泡排序(BubbleSort)算是排序最简单的，可能比桶排序要一丢丢的难度，
描述： 假如数组a 长度为n 需要进行 n-1轮比较
每轮比较
    循环从0到(n-轮次-1) 并与下一个元素比较 如果符合差异条件 则交换

伪代码：
输入：一个长度为n的待排数组a
    遍历 1 → (n-1) (当前轮次为i)
        遍历 0 → a[n-i-1] (当前循环为j)
            根据大小规则比较 a[j] a[j+1] (升序或者降序 冒泡一般位升序)
            如果 a[j] > a[j+1] 则交换 a[i] 与 a[j]
            如果 a[j] <= a[j] 什么都不做
        j 循环结束
    i循环结束






array = []int// 待排序数组
length = len(a) // a的数组长度
for (i from 1 to length-1) {
    for (j from 0 to (length - i - 1) ) {
        if (a[j] > a[j+1]) {
            tmp = a[j+1]
            a[j+1] = a[j]
            a[j] = tmp
        }
    }
}
